# Мои первые шаги в C#

Я начинаю изучать логику программирования. пример того, как программа общается с пользователем:

```csharp
Console.WriteLine("Как твое имя?"); 
// Программа выводит текст на экран.

string? name = Console.ReadLine(); 
// Программа ждет ввода и сохраняет его в переменную name.

Console.WriteLine($"Привет, {name}!"); 
// Программа подставляет имя из переменной в приветствие.
